Fizzbuzz Project

Week 3

Design

My JavaScript code was relatively straightforward. I first wrote a loop that iterated from 1 to 200, which contained a call to document.write(). If the index of the loop was a multiple of 3, 5, or 7 (which I tested with modulo), then I appended 'Fizz', 'Buzz', or 'Boom' to a string variable. I didn't use if else for these conditions because they are not mutually exclusive. Then if the variable was empty, print the index of the loop, else print the variable followed by an `!`.

Challenges

I had trouble figuring out where to put my script tag. At first I put it in the head, thinking that document.write() would print to the end of body. This broke the webpage, but I was later able to figure out that document.write() writes to the element in which its script is located.
