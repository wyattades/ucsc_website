Week 4

Design

My JavaScript design consisted of functions to perform three simple tasks: return a random cat fact from a predefined array, return a string of html for a random image of a cat (from `unsplash.me`), and set the innerHtml of the element with id="js-output" with the output of the previous two functions.

Challenges

I didn't have trouble with the majority of the JavaScript. The only difficult portion was figuring out how to load a random image of a cat. After some research, I found that I can use the url `source.unsplash.me` to get a random image! After some trial and error, the correct url for fetching a random image of a cat is:
https://source.unsplash.me/collection/[collectionId]/[width]x[height]?sig=[randomIndex]
where collectionId is the id of a collection of cats
and randomIndex is a number between 0 and the size of the collection.
