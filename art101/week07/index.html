Ajax Info Fetcher

Week 7

Design

My page's design is the same as the previous week's.

My code consists of fetching a data array from an API when the page loads (I only have to do this once because the response never changes). Then, I choose a random element from that array to display in #output every time the button is clicked. Also, the API returns JSON and jQuery automatically parses it into a JavaScript array so the API response is easy to work with.

Challenges

I've worked with APIs and Ajax before, so this assignment wasn't too difficult. The hardest part was definitely finding a cool API that responded with JSON and didn't throw a Access-Control-Allow-Origin error.
